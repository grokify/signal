## [v0.1.0] - 2026-02-22

### Highlights

- Planet-style blog aggregator with JSON Feed 1.1 output designed for AI agents and human readers
- Agent-friendly API structure with predictable URLs, schema.json, and AGENTS.md
- Priority links for hand-curated content with discussion links and image pins

### Added

- JSON Feed 1.1 specification types with Signal extensions (`_signal_feed_title`, `_signal_priority`, `_signal_discussions`, `_signal_source`) ([`79b787a`](https://github.com/grokify/signal/commit/79b787a))
- Core entry types with deduplication, sorting, filtering, and JSON Feed conversion ([`79b787a`](https://github.com/grokify/signal/commit/79b787a))
- OPML in JSON format for feed source management ([`b12db8a`](https://github.com/grokify/signal/commit/b12db8a))
- Priority links package for hand-curated content with discussions (HackerNews, Reddit, Lobsters) and image pins ([`b12db8a`](https://github.com/grokify/signal/commit/b12db8a))
- Monthly file splitting with merge mode to preserve historical entries ([`b12db8a`](https://github.com/grokify/signal/commit/b12db8a))
- Feed aggregator with concurrent RSS/Atom fetching and progress reporting ([`54da899`](https://github.com/grokify/signal/commit/54da899))
- Agent-friendly API generation with schema.json, AGENTS.md, and organized endpoints (by-month, by-source, by-tag) ([`b138d82`](https://github.com/grokify/signal/commit/b138d82))
- Atom feed output generation for RSS reader compatibility ([`b138d82`](https://github.com/grokify/signal/commit/b138d82))
- Cobra-based CLI with `aggregate` command supporting OPML input, priority links, monthly splitting, and API generation ([`9567ba3`](https://github.com/grokify/signal/commit/9567ba3))
- Source platform metadata support for LinkedIn, Twitter, and other platforms ([`79b787a`](https://github.com/grokify/signal/commit/79b787a))
- Discussion links support for HackerNews, Reddit, and Lobsters with score and comment counts ([`79b787a`](https://github.com/grokify/signal/commit/79b787a))
- Image pins for visual content (LinkedIn posts, articles with hero images) ([`79b787a`](https://github.com/grokify/signal/commit/79b787a))
